## ğŸ”§ Troubleshooting

### Common Issues

**Problem:** Audio/TTS not working
- **Solution:** Check browser audio permissions, try Chrome browser
- **Alternative:** Click sounds can be enabled in settings (top right)

**Problem:** Student progress lost
- **Solution:** Progress saves to browser localStorage. If browser data is cleared, progress is lost
- **Prevention:** Don't clear browser data during pilot

**Problem:** Assessment not showing
- **Solution:** Clear browser cache, try incognito mode
- **Check:** Look for `assessmentCompleted` in browser console

**Problem:** App looks broken
- **Solution:** Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
- **Check:** Ensure JavaScript is enabled

**Problem:** Student placed at wrong level
- **Note:** Assessment is adaptive. If student seems misplaced:
  - Check their actual performance in modules
  - Levels can be adjusted manually (requires code change)
  - Document for post-pilot review
